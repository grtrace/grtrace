List of all supported commands in the scene description language

Documentation format:
(required) [optional]
* - object of any type
Types:
S - string
V2, V3 - vectors
M4 - 4x4 matrix
N - number (float)
W - wavelength (float)
C - color
T - texture

Type cnstructors:
VEC2 X Y
VEC3 X Y Z
MAT4 A B C D  E F G H  I J K L  M N O P
FLOAT N
WAVE N
COLOR RRGGBB - in HTML/hexadecimal format
